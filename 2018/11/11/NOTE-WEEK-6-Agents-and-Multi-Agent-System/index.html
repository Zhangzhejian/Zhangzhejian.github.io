Lecture 6: Negotiation(reaching agreements)

Game theory

Basic knowledge

  • Dominant Strategy
  • Nash Equilibrium
  • Pareto optimal
  • social welfare

Properties of agreement mechanism

  • An agreement mechanism is individually rational if no agent will end up worse off as a result of participating (agents cannot end up with negative expected utility). Otherwise it would not be rational for agents to participate.

  • An agreement mechanism is symmetric if the rules of the game are the same for all participants.

  • An agreement mechanism is fair if participants have equal opportunity to influence the outcome.

  • An agreement mechanism is Pareto optimal if it guarantees a Pareto optimal outcome.

  • An agreement mechanism maximises social utility if it guarantees an outcome that maximises social utility.

  • An agreement mechanism is guaranteed to terminate if there is no way that participants following the protocol can continue to interact indefinitely.

  • An agreement mechanism is stable if a Nash Equilibrium exists.

  • An agreement mechanism is simple if it is easy to work out what the best strategy is.

Monotonic concession protocol(单方面让步协议)

Negotiation proceeds in rounds.

On round 1, agents simultaneously propose a deal from the negotiation set.

Agreement is reached if one agent finds that the deal proposed by the other is at least as good or better than its proposal. If each agent’s offer matches or exceeds that of the other agent, then one of the proposals is randomly selected as the agreement deal. If only one proposal exceeds of matches the other’s, then this is the agreement deal.

If no agreement is reached, then negotiation proceeds to another round of simultaneous proposals.

In round u + 1, no agent is allowed to make a proposal that is less preferred by the other agent than the deal it proposed at time u.

If neither agent makes a concession in some round u > 0, then negotiation terminates, with the conflict deal.

Zeuthen strategy

The Zeuthen strategy determines what an agent should do by answering three questions.

  • What should an agent’s first proposal be?

    Its most preferred deal.

  • On any given round, who should concede?

    The agent least willing to risk conflict (if equal risk, flip a coin).

  • If an agent concedes, then how much should it concede?

    Just enough to change the balance of risk.

  • Phantom and Decoy tasks

  • Hidden tasks

    • Pretending not to have been allocated tasks that you have been
