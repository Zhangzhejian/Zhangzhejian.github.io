Cross validation(交叉验证）

s k-fold cross validation.

  1. Split data into k equal subsets. Learn on k-1 sets and test each result on the remainder.

  2. Repeat k times.

  3. Average test set score is a better estimate of the error rate than a single score.

  4. Common values of k are 5 and 10, both giving error estimates that are very likely to be accurate.

Clustering

  1. Given a set of examples:

​ identify K different groups.

  1. Each $x_i$ should be in the group that it matches most closely. Identify each group by its centre/mean, $u_j$ .

  2. The clusters $z^i$ are then made up of the points closest to the $µ_j $.

K-means

  • Pick random values for µ k .

  • Then repeat:

    1. Assign each x i to its closest cluster centre:

    2. Update each cluster centre as the mean of the points assigned to that cluster:

      until converged
