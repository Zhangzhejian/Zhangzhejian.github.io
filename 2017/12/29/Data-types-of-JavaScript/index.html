Six basic types of data in JavaScript

1. NULL

The value of the type is null which points to an empty object, thus when use typeof to detect the type of a null value, an object will be returned.

2. Boolean

Boolean is also a special type in JS. There are only two values for boolean type, true or false. In addition, boolean is case sensitive, which means True and False are different from true and false.

When trying to convert other types to Boolean using Boolean():

Type Equals true Equals false
Boolean true false
String any other non-null value “”(null String)
Number any non-zero value 0 or NaN(not a number)
Object any object null(as mentioned above, null is an object)
Undefined not applicable(always false) undefined
