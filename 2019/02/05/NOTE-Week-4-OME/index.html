Dijkstra’s Algorithm

All the weights should be non-negative.

  • EXTRACT-MIN(Q) is the function that find a vertex with minimum value of v.d
  • $G.Adj(u)$ is the function returns the neighbors of the vertex u
  • Complexity: $O(ELog(V))$ Quora
    • V = N, E = N(N-1)
    • the complexity of EXTRACT-MIN function is $O(logV)$
    • the whole complexity is $O(V(logV+V-1)) = O((E+V)logV)$
    • in the worst case, E>>V, so the complexity is $O(ElogV)$
