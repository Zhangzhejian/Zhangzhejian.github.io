Web Application Specification

Posted on 2019-02-05

Content:

  • Web applications and when to use them.
  • Structure of a web application.
  • Web development.
  • Portability, usability, and accessibility.
  • Web page design.
  • Client-side vs. server-side processing.

When to use Web application

  • Involve access to remote data
  • do not require fine grained exchange of data
    • 不需要细粒度的数据交换（基于web的数据交换是比较昂贵的）
  • do not require highly time critical responses
    • 信息交互有较大延迟
  • receive and return data which can be effectively transmitted across the internet
    • 信息可以高效的在互联网中传播

There are many rapidly-evolving technologies in web application area, which requires the web applications to be flexible and easily upgraded.And Web applications often have common structure and elements.

Protability, Usability, Accessibility

  • Protability(可移植性、通用性)
    • system can be moved to different execution environments and provide the same functionality in the new environment as in the old.
    • UI in different browser
    • Difference between mobile and PC
  • Usability(使用体验)
    • web interface does not require unreasonable effort to use, and that users can access provided functionality without excess effort
    • UI:
      • Clear information
      • feedback
      • related functions
    • program logic design
  • Accessibility(可用性)
    • a web interface can be used by users of differing ability — such as visually impaired, colour-blind, deaf, or senior citizen users — as effectively as by non-disabled, technological skilled users.
    • easy to understand

Client-Server Comparation

Consideration:

  1. Computational Load
  2. Security issues

Client-side

  1. Data validation and dynamic user feedback

Server-side processing

  1. Processing data sent from the client side involving critical checks on the correctness of the data and security checks(认证，授权) 安全考虑
  2. Modifying or retrieving data in lower tiers of the server side, such as a database 安全考虑
  3. Invoking operations of lower tiers, including remote web services 安全考虑
  4. Generating content to be shown to the client 计算负载考虑
