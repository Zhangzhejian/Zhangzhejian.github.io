(Blockchain)Introduction to Cryptocurrencies

Posted on 2019-01-19

Blockchain provides verification of the sequence of transactions to re-create a database, but it doesn’t provide a mechanism to prove its the same database everyone else is using.

Consensus on a global scale

A set of computers want to reach agreement that about a single decision (or data item)

Consideration

  • Safety
    • Everyone should agree and never forget
  • Liveness
    • How many peers must be online before a decision is final

Classical Consensus

  • small scale and up to 1/3 fault tolerance
  • safety and fork-avoidance
  • no financial incentive

Nakamoto Consensus

  • Global Scale and 99%+ fault-tolerance
  • Liveness and fork-tolerance(eventual consistency)
    • all computers will eventually agree on a decision
  • Financial incentive(Profit)
    • Rewarded with coins for participating

Bitcoin is the first consensus protocol that strives for liveness instead of safety.

All financially motivated peers(miners) will eventually agree upon a single blockchain.
