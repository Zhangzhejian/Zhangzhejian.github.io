Lec 8: Auctions

Key points:

  • English auctions
  • dutch auctions
  • first price sealed bid auctions
  • vickrey auctions
  • combinatorial auctions
    • Vickrey-clarke-groves mechanism

Introduction

Auctions are used to allocate scarce resources that are desired by more than one agent

Auctioneer aims to allocate a good to one of the bidders.

Auctioneer wants to maximise the selling price. Attempts to do this through the design of an appropriate auction.

Bidders want to minimise the price at which the good is auctioned. Attempt to do this by using an appropriate strategy.

  • Public/common value
  • Private value
  • correlated value
  • open cry
  • sealed bid
  • ascending
  • descending
  • one shot
  • first price
  • Second price: highest wins the good and pays the second highest number

English Auction

  • For allocation of a single item.

  • First price, open cry, ascending.

Dutch auction

  • For allocation of a single item.

  • First price, open cry, descending.

First-price sealed bid auctions

  • For allocation of a single item.

  • First price, sealed bit, one shot.

Vickrey Auction

  • For allocation of a single item.

  • Second price, sealed bit, one shot.

Collusions(合谋)

all the bidders can form a coalition (the grand coalition) and agree to buy the item for a low price from the auctioneer. The coalition can then sell the item for a higher price and split the profit between themselves.
