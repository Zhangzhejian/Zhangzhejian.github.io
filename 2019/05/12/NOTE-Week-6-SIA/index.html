Five-tier architecture

  1. Client tier
    • Web pages
  2. Presentation tier
    • controller
    • View components
  3. Business tier
    • session beans
    • entity beans
  4. Integration tier
    • Databse interfaces
    • External Service interface
  5. Resource tier
    • Databases
    • Web services
    • Other resources

Functional core of web applications

  • web pages
    • HTML
  • Controller/coordinator components
    • Servlet
  • View/presentation server-side elements
    • JSP
  • Resources
    • Database, static files

Approch to the presentation tier

JSE:

  • Pure Servlet
  • Pure JSP
  • MVC(Model-view-controller)
    • hybrid approach
    • Servlet(Controllers) handle requests, interact with business tier beans(Models) and create beans for use by JSPs(Views)
