EJB

Enterprise Java Beans (EJBs) are a core mechanism for carrying out business logic on the server side of a J2EE-based system.

Two forms of EJB:

  • Entity beans
  • session beans

EJB interfaces

  • Remote Business interface
    • Business operations
    • setPrice()
  • Local business interface
  • Remote home interface
    • Create()
    • Remove()
    • Find()
  • Local home interface

EJB life cycle

CMP(container-managed persistence) is an automated synchronisation of the entity bean data and the actual stored data in the resource tier provided by J2EE.
