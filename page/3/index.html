Properties of agreement mechanism

  • An agreement mechanism is individually rational if no agent will end up worse off as a result of participating (agents cannot end up with negative expected utility). Otherwise it would not be rational for agents to participate.

  • An agreement mechanism is symmetric if the rules of the game are the same for all participants.

  • An agreement mechanism is fair if participants have equal opportunity to influence the outcome.

  • An agreement mechanism is Pareto optimal if it guarantees a Pareto optimal outcome.

  • An agreement mechanism maximises social utility if it guarantees an outcome that maximises social utility.

  • An agreement mechanism is guaranteed to terminate if there is no way that participants following the protocol can continue to interact indefinitely.

  • An agreement mechanism is stable if a Nash Equilibrium exists.

  • An agreement mechanism is simple if it is easy to work out what the best strategy is.

Monotonic concession protocol(å•æ–¹é¢è®©æ­¥åè®®)

Negotiation proceeds in rounds.
