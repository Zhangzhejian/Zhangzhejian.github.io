Introduction

Auctions are used to allocate scarce resources that are desired by more than one agent

Auctioneer aims to allocate a good to one of the bidders.

Auctioneer wants to maximise the selling price. Attempts to do this through the design of an appropriate auction.

Bidders want to minimise the price at which the good is auctioned. Attempt to do this by using an appropriate strategy.

  • Public/common value
  • Private value
  • correlated value
  • open cry
  • sealed bid
  • ascending
  • descending
  • one shot
  • first price
  • Second price: highest wins the good and pays the second highest number

  • For allocation of a single item.

  • First price, open cry, ascending.

Dutch auction

  • For allocation of a single item.

  • First price, open cry, descending.

First-price sealed bid auctions

  • For allocation of a single item.

  • First price, sealed bit, one shot.

Vickrey Auction

  • For allocation of a single item.

  • Second price, sealed bit, one shot.

Introduction

Social choice theory is about how to make group decisions.

Specifically, it provides a range of voting protocols that can be used to account for the preferences of a group of agents.

Properties of agreement mechanism

  • An agreement mechanism is individually rational if no agent will end up worse off as a result of participating (agents cannot end up with negative expected utility). Otherwise it would not be rational for agents to participate.

  • An agreement mechanism is symmetric if the rules of the game are the same for all participants.

  • An agreement mechanism is fair if participants have equal opportunity to influence the outcome.

  • An agreement mechanism is Pareto optimal if it guarantees a Pareto optimal outcome.

  • An agreement mechanism maximises social utility if it guarantees an outcome that maximises social utility.

  • An agreement mechanism is guaranteed to terminate if there is no way that participants following the protocol can continue to interact indefinitely.

  • An agreement mechanism is stable if a Nash Equilibrium exists.

  • An agreement mechanism is simple if it is easy to work out what the best strategy is.

Negotiation proceeds in rounds.
