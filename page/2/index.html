Introduction

  • Generate a collection of “weak” classiﬁers from a given dataset.
  • Each week classifier is a simple classifier, e.g., a half-space classifier
  • Combine classifiers to be a final one (averaging or voting)

Bagging(bootstrap aggregating)

  • Uses multiple versions of a training set.
  • Each created by drawing $n’ < n$ samples from $\scr D$ with replacement.(有放回的)
  • Each bootstrap data set is used to train a different weak classiﬁer.
  • Final classiﬁcation decision is based on the vote of each weak classiﬁer.

Procedure

  1. Start with dataset$\scr D$

  2. Generate M dataset $ \scr D_1, \scr D_2, \dots ,\scr D_M$

  3. Learn weak classifier for each dataset

    • $f_i(x)$ for dataset $\scr D_i$

  4. Combine all weak classifiers using a majority voting scheme

Boosting

Goal: to improve the accuracy of any given learning algorithm

Adaboost(Adaptive Boost)

  • Advantages
    • Simple for implementation
    • Good classification accuracy and generalisation
    • Can be applied to different classifiers
  • Disadvantage
    • Solution is suboptimal(次优)
    • sensitive to noisy data and outliers(异常点)

Algorithm:

  1. http://www.52caml.com/head_first_ml/ml-chapter6-boosting-family/
  • First create a classiﬁer with accuracy on the training set greater than average.
  • Then add new component classiﬁers to form an ensemble whose joint decision rule has arbitrarily high accuracy on the training set.

Stacked generalization

  • Two levels(Input and output)
  • The input will be processed by a group of classiﬁers in the ﬁrst level
  • The output from the ﬁrst layer will go though the classiﬁer in the second level to make the ﬁnal decision
  1. Prepare K sub-training dataset: $\scr D_1 , \scr D_2 , ··· , \scr D_K$ of different size and samples
  2. Train the k th classiﬁer using training dataset $\scr D_k $
  3. Create a new training dataset for the second-level classiﬁer. The outputs of the ﬁrst-level classiﬁers ($z_1 , z_2 , ··· , z_K$ ) and the target output (y) of the classiﬁcation problem (supervised) are taken to form the new dataset $D_{new}$
  4. Train the second-level classiﬁer using the dataset $D_{new}$
